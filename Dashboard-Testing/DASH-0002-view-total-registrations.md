# DASH-0002: Dashboard testing - View Total Registrations

> Summary: Verify that the dashboard displays the total number of chainsaw registrations.

**Preconditions:** The tester is logged in to Chainventory using a valid account.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Launch the Chainventory website. | Verify that the login page or dashboard is shown. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Observe the dashboard summary cards. | Verify that the total registrations card is visible. |
| 4 | Check the displayed total registrations value. | Verify that the value is shown clearly and is not blank. |
| 5 | Compare the displayed value with the available chainsaw records, if applicable. | Verify that the total registrations count matches the records available to the user. |

**Post-conditions:**

* The dashboard displays the total number of registrations.
* The total registrations count is readable and accessible to the user.
