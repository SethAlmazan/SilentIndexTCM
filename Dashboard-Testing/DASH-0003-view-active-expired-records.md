# DASH-0003: Dashboard testing - View Active and Expired Records

> Summary: Verify that the dashboard displays the number of active and expired chainsaw records.

**Preconditions:** The tester is logged in to Chainventory using a valid account.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Launch the Chainventory website. | Verify that the login page or dashboard is shown. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Observe the dashboard summary cards. | Verify that the active and expired records cards are visible. |
| 4 | Check the active records value. | Verify that the number of active records is displayed clearly and is not blank. |
| 5 | Check the expired records value. | Verify that the number of expired records is displayed clearly and is not blank. |
| 6 | Compare the values with the records list, if applicable. | Verify that the active and expired counts match the available chainsaw records. |

**Post-conditions:**

* The dashboard displays active and expired record counts.
* The active and expired values are readable and understandable to the user.
