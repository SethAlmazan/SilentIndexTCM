# RES-0002: Responsive testing - Registration Form Mobile View

> Summary: Verify that the chainsaw registration form displays properly on mobile screen sizes.

**Preconditions:** The tester is logged in to Chainventory using a valid account and is using a mobile device or browser mobile view.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website on a mobile device or browser mobile view. | Verify that the website loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Navigate to the `Registration` page. | Verify that the registration form opens successfully. |
| 4 | Observe the registration form layout. | Verify that the form fields are properly stacked and not cramped. |
| 5 | Scroll through the registration form. | Verify that all form sections are accessible. |
| 6 | Check the input fields and labels. | Verify that the fields and labels are readable on mobile view. |
| 7 | Check the image upload sections. | Verify that the upload areas and image previews fit properly on the screen. |
| 8 | Enter sample information in the form. | Verify that the form can be filled out properly on mobile view. |

**Post-conditions:**

* The registration form is usable on mobile view.
* Form fields, labels, and upload sections are readable and properly arranged.
