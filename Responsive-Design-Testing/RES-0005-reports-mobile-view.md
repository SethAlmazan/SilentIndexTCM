# RES-0005: Responsive testing - Reports Mobile View

> Summary: Verify that the reports page displays properly on mobile screen sizes.

**Preconditions:** The tester is logged in to Chainventory using a valid account and is using a mobile device or browser mobile view.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website on a mobile device or browser mobile view. | Verify that the website loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Navigate to the `Reports` page. | Verify that the reports page opens successfully. |
| 4 | Observe the report summary cards. | Verify that total registrations, active permits, expired permits, and compliance rate are readable. |
| 5 | Check the reports chart or pie graph. | Verify that the graph fits properly on the mobile screen. |
| 6 | Scroll through the reports page. | Verify that all report sections are accessible without layout issues. |
| 7 | Check the navigation menu or buttons. | Verify that navigation remains usable on mobile view. |

**Post-conditions:**

* The reports page is usable on mobile view.
* Report cards and graphs are readable and properly arranged.
