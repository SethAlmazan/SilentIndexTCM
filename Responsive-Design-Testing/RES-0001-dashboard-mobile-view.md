# RES-0001: Responsive testing - Dashboard Mobile View

> Summary: Verify that the dashboard page displays properly on mobile screen sizes.

**Preconditions:** The tester is logged in to Chainventory using a valid account and is using a mobile device or browser mobile view.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website on a mobile device or browser mobile view. | Verify that the website loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Observe the dashboard layout. | Verify that the dashboard content fits properly within the mobile screen. |
| 4 | Check the dashboard cards or summary sections. | Verify that the cards are readable and not overlapping. |
| 5 | Scroll through the dashboard page. | Verify that the page scrolls smoothly and all content is accessible. |
| 6 | Check the navigation menu or buttons. | Verify that navigation remains usable on mobile view. |

**Post-conditions:**

* The dashboard is usable on mobile view.
* Dashboard content is readable and properly arranged.
