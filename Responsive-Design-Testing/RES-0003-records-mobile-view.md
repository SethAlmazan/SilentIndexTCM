# RES-0003: Responsive testing - Chainsaw Records Mobile View

> Summary: Verify that the chainsaw records page displays properly on mobile screen sizes.

**Preconditions:** The tester is logged in to Chainventory using a valid account, at least one chainsaw record exists, and the tester is using a mobile device or browser mobile view.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website on a mobile device or browser mobile view. | Verify that the website loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Navigate to the `Records` page. | Verify that the chainsaw records page opens successfully. |
| 4 | Observe the records table layout. | Verify that the table is displayed properly on mobile view. |
| 5 | Scroll horizontally through the records table. | Verify that all columns can be accessed without breaking the layout. |
| 6 | Check the `Owner Name` column. | Verify that the owner name column remains visible or usable on mobile view. |
| 7 | Check the action buttons. | Verify that view, edit, delete, and export actions are accessible on mobile view. |
| 8 | Search or filter records. | Verify that search and filter functions remain usable on mobile view. |

**Post-conditions:**

* The chainsaw records page is usable on mobile view.
* The records table remains readable and accessible on smaller screens.
