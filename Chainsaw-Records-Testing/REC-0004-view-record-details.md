# REC-0004: Records testing - View Record Details

> Summary: Verify that the user can open and view the complete details of a selected chainsaw record.

**Preconditions:** The tester is logged in to Chainventory using a valid account and at least one chainsaw record exists.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website. | Verify that the system loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Click the `Records` navigation link or button. | Verify that the chainsaw records page opens. |
| 4 | Locate an existing chainsaw record in the table. | Verify that the record is visible in the records list. |
| 5 | Click the `View` or eye icon action for the selected record. | Verify that the record details modal or page opens. |
| 6 | Review the owner information. | Verify that the owner name, address, contact number, and email are displayed correctly. |
| 7 | Review the chainsaw information. | Verify that brand, model, serial number, length, power rating, description, origin/source, and price are displayed correctly. |
| 8 | Review the registration information. | Verify that registration number, purpose, area/location, registration date, expiry date, status of issuance, and status are displayed correctly. |
| 9 | Check the uploaded images section. | Verify that proof of ownership and inspection images are displayed, if available. |
| 10 | Close the record details modal or page. | Verify that the user returns to the chainsaw records list. |

**Post-conditions:**

* The selected chainsaw record details are displayed successfully.
* The user can return to the records list after viewing the details.
