# REC-0001: Records testing - View Chainsaw Records List

> Summary: Verify that the user can view the list of registered chainsaw records.

**Preconditions:** The tester is logged in to Chainventory using a valid account and at least one chainsaw record exists.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website. | Verify that the system loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Click the `Records` navigation link or button. | Verify that the chainsaw records page opens. |
| 4 | Observe the records table. | Verify that the table is displayed properly. |
| 5 | Check the visible record columns. | Verify that owner name, address, chainsaw details, dates, status, and actions are visible. |
| 6 | Check if existing records are listed. | Verify that saved chainsaw records appear in the table. |
| 7 | Scroll horizontally, if needed. | Verify that the table remains usable and readable. |

**Post-conditions:**

* The chainsaw records list is displayed successfully.
* The user can view saved chainsaw registration records.
