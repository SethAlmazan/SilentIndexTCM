# REP-0001: Reports testing - View Reports Page

> Summary: Verify that the user can open and view the reports page successfully.

**Preconditions:** The tester is logged in to Chainventory using a valid account.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website. | Verify that the system loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Click the `Reports` navigation link or button. | Verify that the reports page opens successfully. |
| 4 | Observe the reports page layout. | Verify that the reports page content is displayed properly. |
| 5 | Check the report summary cards. | Verify that report data such as total registrations, active permits, expired permits, and compliance rate are visible. |
| 6 | Check the graph or chart section, if available. | Verify that the reports graph is displayed properly. |

**Post-conditions:**

* The reports page is displayed successfully.
* The user can view report summaries and visual report data.
