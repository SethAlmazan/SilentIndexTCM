# REP-0005: Reports testing - View Reports Pie Graph

> Summary: Verify that the reports page displays the permit status pie graph correctly.

**Preconditions:** The tester is logged in to Chainventory using a valid account and the reports page has available report data.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Open the Chainventory website. | Verify that the system loads successfully. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Click the `Reports` navigation link or button. | Verify that the reports page opens successfully. |
| 4 | Locate the pie graph or chart section. | Verify that the pie graph section is visible. |
| 5 | Observe the displayed chart. | Verify that the chart loads properly and is not broken or blank. |
| 6 | Check the chart labels. | Verify that the chart represents active and expired permit data clearly. |
| 7 | Compare the chart with the report summary values, if applicable. | Verify that the chart data matches the report summary counts. |

**Post-conditions:**

* The reports pie graph is displayed successfully.
* The graph clearly represents the permit status data.
