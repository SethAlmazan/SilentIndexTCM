# AUTH-0001: Authentication testing - Register Account

> Summary: Verify that a new user account can be registered successfully.

**Preconditions:** The tester is on the Chainventory deployed website and is not logged in.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Launch the Chainventory website. | Verify that the login page is shown. |
| 2 | Click the `Register` link or button. | Verify that the registration page is shown. |
| 3 | Enter a valid email address. | Verify that the email field accepts the input. |
| 4 | Enter a valid password. | Verify that the password field accepts the input. |
| 5 | Click the `Register` or `Sign Up` button. | Verify that the account registration process is submitted. |
| 6 | Wait for the system response. | Verify that the system successfully creates the account or shows a proper confirmation message. |

**Post-conditions:**

* A new user account is created.
* The user can proceed to login using the registered account.
