# DASH-0001: Dashboard testing - View Dashboard Overview

> Summary: Verify that the dashboard overview is displayed successfully after login.

**Preconditions:** The tester is logged in to Chainventory using a valid account.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Launch the Chainventory website. | Verify that the login page or dashboard is shown. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Observe the dashboard page. | Verify that the dashboard overview is displayed. |
| 4 | Check the dashboard cards or summary sections. | Verify that important system information is visible. |
| 5 | Check the navigation menu. | Verify that links to records, registration, and reports are available. |

**Post-conditions:**

* The dashboard is displayed successfully.
* The user can navigate to other Chainventory modules.+
