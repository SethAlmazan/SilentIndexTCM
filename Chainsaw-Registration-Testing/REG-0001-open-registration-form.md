# REG-0001: Registration testing - Open Registration Form

> Summary: Verify that the chainsaw registration form can be opened successfully from the navigation menu.

**Preconditions:** The tester is logged in to Chainventory using a valid account.

**Scenario 1**

| # | Step | Expected Behavior |
|---|------|-------------------|
| 1 | Launch the Chainventory website. | Verify that the login page or dashboard is shown. |
| 2 | Log in using a valid account, if needed. | Verify that the user is redirected to the dashboard. |
| 3 | Locate the `Registration` navigation link or button. | Verify that the registration option is visible. |
| 4 | Click the `Registration` navigation link or button. | Verify that the chainsaw registration page opens. |
| 5 | Observe the registration form. | Verify that the registration form fields are displayed properly. |
| 6 | Check the form layout. | Verify that the form is readable and properly arranged. |

**Post-conditions:**

* The chainsaw registration form is displayed successfully.
* The user can start entering chainsaw registration details.
